# Getting Started

Bloxstrap is in early development. The project structure is being established.

## Prerequisites

- Node.js 16+
- npm 8+
- Git

## Installation

```bash
git clone https://github.com/NexusAI-lovable/Bloxstrap.git
cd Bloxstrap
npm install
```

## Running Tests

```bash
npm test
```

## File Structure

```
src/        - Source code
tests/      - Tests
docs/       - Documentation
config/     - Configuration
assets/     - Assets
.github/    - GitHub templates
```

See README.md for more information.
